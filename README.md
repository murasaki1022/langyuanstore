### **使用技術**

- 以 `Vue 3` 與 `create-vue`架設開發。
- 使用 `Bootstrap 5` 作為 CSS 框架切版。
- 使用 `ESLint standard` 統一程式碼結構。
- 使用 `Pinia.js` 傳遞跨元件資料，作為狀態管理工具。
- 使用 `Axios` 串接API。

### **使用套件**

- `Vue 3`
- `Create-vue`
- `Vue Router`
- `Axios`
- `Vue-loading-overlay`
- `Vee Validation`
- `Pinia`
- `Bootstrap 5`
- `Swiper`
- `SweetAlert2`
- `Aos`
